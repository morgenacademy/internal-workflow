0. Eenmalig instellen (belangrijk)

Dit moet iedereen één keer doen voordat de workflow werkt.

1
Installeer Git: git-scm.com
2
Maak een map voor projecten bijvoorbeeld ~/Repos bijvoorbeeld C:\Repos
3
Clone de GitHub repo.
Clone repo (Mac) 
cd ~/Repos
git clone REPO-URL project
Clone repo (Windows) 
cd C:\Repos
git clone REPO-URL project

1. Save shortcut installeren

Dit zorgt dat het commando save werkt.

Open config (Mac) 
nano ~/.zshrc
Voeg toe 
alias save="git add . && git commit -m 'update' && git push"
Herlaad terminal 
source ~/.zshrc

2. Terminal openen in de juiste map

Dit is de stap waar beginners vaak vastlopen.

1
Open Finder.
2
Ga naar je projectmap.
3
Rechtsklik → Open Terminal at Folder.
1
Open Verkenner.
2
Ga naar je projectmap.
3
Klik adresbalk → typ powershell.

3. Begin van je werkdag

Update project 
git pull

4. Werk opslaan

Save 
save
